# How to use the macro

## Copy macro code

Copy the macro code from the caption.gs file to your AppsScript editor of your google docs file.

You can find the AppsScript editor in the docs menu. Save the script to your document.

<img width="1151" height="216" alt="macro_user_guide_0" src="https://github.com/user-attachments/assets/110097d9-28a2-4d71-acb1-be919ba07705" />

## Execute macro via newly added menu

Reopen your document. A newly added menu should be visible. You can now execute the macro via the button.
